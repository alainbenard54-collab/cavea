# ☑️ Sélection multiple

Sélectionner plusieurs bouteilles pour les déplacer ou les consommer en une seule opération.

## Prérequis

- Application ouverte en mode écriture
- La sélection multiple n'est pas disponible en mode 🔒 lecture seule

## Étapes

### 1. Activer le mode sélection

**Appui long** sur une bouteille dans la vue **🍷 Stock** ou dans la vue **📦 Emplacements**. La ligne passe en mode sélectionné (case cochée ☑️, fond coloré). La barre d'actions apparaît en bas de l'écran.

### 2. Sélectionner d'autres bouteilles

- **Tap simple** sur d'autres lignes pour ajouter/retirer de la sélection
- Sur desktop (tableau), une colonne ☑️ apparaît en première position
- Le compteur dans la barre d'actions affiche "N bouteille(s) sélectionnée(s)"

### 3. Choisir une action

**↕️ Déplacer**

Ouvre un panneau où vous saisissez l'emplacement de destination (avec autocomplétion). Toutes les bouteilles sélectionnées sont déplacées vers cet emplacement en une seule transaction.

**🍸 Consommer**

Ouvre un panneau où vous saisissez la date, la note /10 (optionnelle) et le commentaire (optionnel). Toutes les bouteilles sélectionnées sont consommées avec les mêmes informations en une seule transaction.

### 4. Annuler la sélection

- Appuyez sur **✕ Annuler** dans la barre d'actions
- Ou naviguez vers un autre écran (la sélection est automatiquement vidée)
