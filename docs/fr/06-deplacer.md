# ↕️ Déplacer une bouteille

Changer l'emplacement d'une bouteille sans la consommer.

## Prérequis

- Application ouverte en mode écriture

## Étapes

### 1. Ouvrir le panneau d'actions

Dans la vue **🍷 Stock**, appuyez sur la ligne de la bouteille à déplacer.

### 2. Choisir ↕️ Déplacer

Appuyez sur **↕️ Déplacer** dans le panneau d'actions.

### 3. Saisir le nouvel emplacement

Le champ d'emplacement propose une autocomplétion sur tous les emplacements de votre cave (stock ET consommées). Tapez les premières lettres pour filtrer les suggestions.

**Format d'emplacement** : `Niveau1` ou `Niveau1 > Niveau2 > Niveau3`

Exemples : `Cave`, `Cave > Casiers > B3`, `Réserve`

Vous pouvez aussi saisir un emplacement entièrement nouveau — il sera créé automatiquement.

### 4. Confirmer

Appuyez sur **Déplacer**. Seul le champ `emplacement` est mis à jour. La bouteille reste en stock.

> **↕️ Déplacer ≠ 🍸 Consommer** : changer l'emplacement ne marque pas la bouteille comme consommée. La date de sortie n'est pas touchée.
