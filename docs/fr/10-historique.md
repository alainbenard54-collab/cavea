# 🕐 Historique des consommations

Consulter la liste des bouteilles consommées, rechercher, voir le détail et réhabiliter si besoin.

## Étapes

### 1. Ouvrir l'onglet Historique

Appuyez sur l'onglet **🕐 Historique** dans la navigation. La liste affiche toutes les bouteilles consommées, triées par date de consommation décroissante (la plus récente en premier).

### 2. Rechercher dans l'historique

Utilisez la barre de recherche pour filtrer par domaine, appellation, millésime ou tout autre texte. La recherche est instantanée.

### 3. Consulter le détail d'une bouteille

Appuyez sur une bouteille pour ouvrir le panneau de détail, qui affiche :
- Toutes les informations de la fiche
- La date de consommation, la note /10 et le commentaire de dégustation
- Un badge de maturité calculé à la date de consommation (🔵🟢🔴)

### 4. Réhabiliter une bouteille

Si vous avez consommé une bouteille par erreur, appuyez sur **Réhabiliter** dans le panneau de détail.

La réhabilitation :
- Efface la date de consommation (`date_sortie`)
- Efface la note et le commentaire de dégustation
- Remet la bouteille dans le **🍷 Stock** avec son emplacement d'origine

> Disponible en mode écriture uniquement.
