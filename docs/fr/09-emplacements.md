# 📦 Navigation par emplacement

Explorer votre cave par zone géographique, consulter les statistiques par emplacement et agir directement sur les bouteilles.

## Étapes

### 1. Ouvrir l'onglet Emplacements

Appuyez sur l'onglet **📦 Emplacements** dans la navigation (troisième position).

L'écran affiche l'arbre des emplacements de votre cave.

### 2. L'arbre des emplacements

Chaque nœud représente un niveau d'emplacement. L'arbre est construit automatiquement à partir des emplacements de toutes vos bouteilles en stock.

Chaque nœud affiche :
- Le nom du niveau
- Le nombre de bouteilles (incluant les sous-niveaux)
- La valeur totale (incluant les sous-niveaux) — avec mention "dont K sans prix" si certaines bouteilles n'ont pas de prix

### 3. Naviguer dans l'arbre

Appuyez sur un nœud pour entrer dans ce niveau. Le **fil d'ariane** en haut de l'écran indique votre position (ex : `Emplacements > Cave > Casiers`).

Cliquez sur n'importe quel segment du fil d'ariane pour remonter directement à ce niveau.

Sur Android, le bouton retour système navigue dans l'arbre avant de remonter à la navigation principale.

### 4. Bouteilles directes et sous-emplacements

Si un emplacement contient à la fois des sous-niveaux ET des bouteilles rattachées directement :
- Les sous-niveaux s'affichent en premier
- Une tuile "Directement dans cet emplacement" permet d'afficher uniquement les bouteilles sans sous-niveau

### 5. Agir sur les bouteilles

Dans la liste de bouteilles d'un emplacement :
- Appuyez sur une bouteille → panneau d'actions (**ℹ️ Consulter**, **🍸 Consommer**, **↕️ Déplacer**, **✏️ Modifier**)
- Appui long → mode sélection multiple ☑️ (**↕️ Déplacer** ou **🍸 Consommer** en lot)

La sélection multiple est désactivée en mode 🔒 lecture seule.
