# 🍷 Consulter le stock

Naviguer dans votre stock, appliquer des filtres et trier les bouteilles.

## Étapes

### 1. Ouvrir la vue stock

L'onglet **🍷 Stock** est l'écran principal de l'application. Il affiche toutes les bouteilles actuellement en cave (date de sortie vide).

### 2. Filtres disponibles

Les filtres se combinent (ET logique) :

| Filtre | Type | Description |
|---|---|---|
| Couleur | Multi-sélect | Cochez une ou plusieurs couleurs |
| Maturité | Multi-sélect | Trop jeune / Optimal / À boire / Sans données |
| Appellation | Texte libre | Filtre "contient" |
| Millésime | Texte libre | Année exacte ou partielle |
| Recherche | Texte libre | Recherche dans domaine, appellation, cru, commentaire |

Sur Android en paysage, les filtres sont collapsibles via un bouton "Filtres" pour agrandir la liste.

### 3. Tri des colonnes (desktop ≥ 640 px)

Sur desktop, le stock s'affiche en tableau. Cliquez sur un en-tête de colonne pour trier. Cliquez à nouveau pour inverser l'ordre.

Les colonnes incluent domaine, appellation, millésime, couleur, emplacement, et la colonne **GARDE** — niveau de maturité coloré avec le delta en années.

### 4. Vue liste (mobile / fenêtre étroite)

Sur mobile ou en fenêtre étroite, le stock s'affiche en liste. Chaque ligne affiche domaine, appellation, millésime et un badge de maturité.

### 5. Accéder aux actions sur une bouteille

Appuyez sur une ligne pour ouvrir le panneau d'actions : **ℹ️ Consulter la fiche**, **🍸 Consommer**, **↕️ Déplacer**, **✏️ Modifier la fiche**.
