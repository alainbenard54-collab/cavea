# 🍸 Consommer une bouteille

Enregistrer la consommation d'une bouteille : date, note et commentaire de dégustation.

## Prérequis

- Application ouverte en mode écriture

## Étapes

### 1. Ouvrir le panneau d'actions

Dans la vue **🍷 Stock**, appuyez sur la ligne de la bouteille à consommer.

> En mode lecture seule 🔒, le panneau affiche uniquement **ℹ️ Consulter la fiche** et **Fermer**.

### 2. Choisir 🍸 Consommer

Appuyez sur **🍸 Consommer** dans le panneau d'actions.

### 3. Remplir le formulaire

| Champ | Obligatoire | Notes |
|---|---|---|
| Date de consommation | Oui | Défaut : aujourd'hui. Modifiable via le sélecteur de date. |
| Note /10 | Non | Note de 0 à 10 |
| Commentaire de dégustation | Non | Vos impressions sur le vin |

> **Déclaration tardive** : si vous oubliez d'enregistrer une consommation le jour même, modifiez la date pour indiquer quand la bouteille a réellement été bue.

### 4. Confirmer

Appuyez sur **Confirmer**. La bouteille disparaît du stock et passe dans l'**Historique** 🕐.

En cas d'erreur, vous pouvez **réhabiliter** la bouteille depuis l'**Historique** 🕐 (scénario [10](10-historique.md)).
