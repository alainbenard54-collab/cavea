# ☑️ Multi-select

Select multiple bottles to move or consume them in a single operation.

## Prerequisites

- App open in write mode
- Multi-select is not available in 🔒 read-only mode

## Steps

### 1. Activate selection mode

**Long-press** a bottle in the **🍷 Stock** view or in the **📦 Locations** view. The row switches to selected state (☑️ checked, coloured background). An action bar appears at the bottom of the screen.

### 2. Select more bottles

- **Single tap** other rows to add or remove them from the selection
- On desktop (table view), a ☑️ column appears in the first position
- The counter in the action bar shows "N bottle(s) selected"

### 3. Choose an action

**↕️ Move**

Opens a panel where you enter the destination location (with autocomplete). All selected bottles are moved to that location in a single transaction.

**🍸 Consume**

Opens a panel where you enter the date, rating /10 (optional), and tasting note (optional). All selected bottles are consumed with the same information in a single transaction.

### 4. Cancel selection

- Tap **✕ Cancel** in the action bar
- Or navigate to another screen (selection is automatically cleared)
