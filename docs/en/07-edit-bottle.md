# ✏️ Edit a bottle

Modify the information on a bottle in stock (domain, appellation, vintage, ageing, etc.).

## Prerequisites

- App open in write mode

## Steps

### 1. Open the action sheet

In the **🍷 Stock** view, tap the row of the bottle you want to edit.

### 2. View details first (optional)

Tap **ℹ️ View details** to see all information in read-only mode before editing.

### 3. Choose ✏️ Edit bottle

Tap **✏️ Edit bottle** in the action sheet. The edit screen opens.

### 4. Edit the fields

All non-protected fields are editable:

- Domain, appellation, vintage, colour, classification, volume
- Purchase price, ageing min/max
- Entry date, supplier, producer
- Entry note

Fields offer autocomplete and filterable dropdowns for reference lists.

### 5. Protected fields (not editable here)

These fields are **not** in the edit form:

| Field | Why protected | How to change |
|---|---|---|
| Consumption date | Set when consuming the bottle | Via **🍸 Consume** |
| Rating /10 | Set when consuming the bottle | Via **🍸 Consume** |
| Tasting note | Set when consuming the bottle | Via **🍸 Consume** |
| Identifier (id) | Primary key — never editable | — |

### 6. Save

Tap the save button. Changes apply immediately.
