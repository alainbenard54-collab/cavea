# 🍷 Browse stock

Navigate your stock, apply filters, and sort bottles.

## Steps

### 1. Open the stock view

The **🍷 Stock** tab is the main screen of the app. It displays all bottles currently in the cellar (no exit date set).

### 2. Available filters

Filters combine together (logical AND):

| Filter | Type | Description |
|---|---|---|
| Colour | Multi-select | Check one or more colours |
| Maturity | Multi-select | Too young / Optimal / Drink now / No data |
| Appellation | Free text | Contains filter |
| Vintage | Free text | Exact or partial year |
| Search | Free text | Searches domain, appellation, classification, notes |

On Android in landscape, filters are collapsible via a "Filters" button to give more room to the list.

### 3. Column sorting (desktop ≥ 640 px)

On desktop, stock is shown as a table. Click a column header to sort. Click again to reverse.

Columns include domain, appellation, vintage, colour, location, and the **AGEING** column — colour-coded maturity level with the year delta.

### 4. List view (mobile / narrow window)

On mobile or in a narrow window, stock is shown as a list. Each row displays domain, appellation, vintage, and a maturity badge.

### 5. Act on a bottle

Tap a row to open the action sheet: **ℹ️ View details**, **🍸 Consume**, **↕️ Move**, **✏️ Edit bottle**.
