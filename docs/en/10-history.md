# 🕐 Consumption history

Browse consumed bottles, search, view details, and reinstate if needed.

## Steps

### 1. Open the History tab

Tap the **🕐 History** tab in the navigation. The list shows all consumed bottles, sorted by consumption date descending (most recent first).

### 2. Search in history

Use the search bar to filter by domain, appellation, vintage, or any other text. Search is instant.

### 3. View bottle details

Tap a bottle to open the detail panel, which shows:
- All bottle information
- Consumption date, rating /10, and tasting note
- A maturity badge calculated at the consumption date (🔵🟢🔴)

### 4. Reinstate a bottle

If you consumed a bottle by mistake, tap **Reinstate** in the detail panel.

Reinstating:
- Clears the consumption date (`exit date`)
- Clears the rating and tasting note
- Returns the bottle to **🍷 Stock** with its original location

> Only available in write mode.
