# 📦 Browse by location

Explore your cellar by geographic zone, view stats per location, and act on bottles directly.

## Steps

### 1. Open the Locations tab

Tap the **📦 Locations** tab in the navigation (third position).

The screen displays the location tree for your cellar.

### 2. The location tree

Each node represents a location level. The tree is built automatically from the locations of all your bottles in stock.

Each node displays:
- The level name
- The bottle count (including sub-levels)
- The total value (including sub-levels) — with a note "of which K without price" when some bottles have no price

### 3. Navigate the tree

Tap a node to enter that level. The **breadcrumb** at the top shows your position (e.g. `Locations > Cellar > Racks`).

Click any segment of the breadcrumb to jump directly to that level.

On Android, the system back button navigates the tree before returning to the main navigation.

### 4. Direct bottles and sub-locations

If a location contains both sub-levels AND bottles attached directly:
- Sub-levels are shown first
- A "Directly in this location" tile lets you view only bottles without a sub-level

### 5. Act on bottles

In the bottle list for a location:
- Tap a bottle → action sheet (**ℹ️ View**, **🍸 Consume**, **↕️ Move**, **✏️ Edit**)
- Long-press → ☑️ multi-select mode (**↕️ Move** or **🍸 Consume** in bulk)

Multi-select is disabled in 🔒 read-only mode.
