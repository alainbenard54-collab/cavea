🔧 Configuration Google Drive

Cavea — Synchronisation entre appareils

Pour la plupart des utilisateurs : si vous avez installé Cavea via l'installateur (CaveaSetup-x.y.z.exe), la configuration est automatique. Vous n'avez rien à faire ici.

Vous avez vu un message "Configuration manquante" ?

Ce message apparaît quand le fichier de configuration Google Drive est absent à côté de l'application. La cause la plus courante est une installation incomplète ou ancienne.

Solution rapide :

  1. Désinstallez Cavea (Panneau de configuration → Programmes → Cavea → Désinstaller)
  2. Téléchargez la dernière version de l'installateur sur la page Releases
  3. Relancez l'installation

Qu'est-ce que le Mode 2 (Google Drive) ?

En Mode 1 (local), votre cave est stockée uniquement sur votre PC Windows. Le Mode 2 permet de partager la même cave entre plusieurs appareils (PC Windows + Android) via votre propre espace Google Drive.

Concrètement :

  • Un dossier Cavea/ est créé dans votre Google Drive
  • Le fichier cave.db y est synchronisé à l'ouverture et à la fermeture
  • Un système de verrou (cave.db.lock) garantit qu'un seul appareil écrit à la fois
  • Aucune donnée ne transite par un serveur tiers — tout passe directement entre votre appareil et votre Google Drive

Comment activer le Mode 2

  1. Ouvrez Cavea → onglet Paramètres
  2. Section "Mode de synchronisation" → cliquer Activer Google Drive
  3. Une fenêtre de connexion Google s'ouvre dans votre navigateur
  4. Connectez-vous avec le compte Google dont vous souhaitez utiliser le Drive
  5. Autorisez Cavea à accéder à son propre dossier Drive
  6. Choisissez si vous voulez envoyer votre cave locale vers Drive, ou récupérer une cave déjà présente sur Drive
Avertissement : en Mode 2, Cavea lit et écrase le fichier cave.db sur Drive à chaque synchronisation. N'activez pas le Mode 2 simultanément depuis deux appareils différents sans utiliser le mécanisme de synchronisation intégré.

Installation manuelle (utilisateurs avancés)

Si vous avez copié cavea.exe sans utiliser l'installateur, vous devez placer le fichier google_desktop_secrets.json dans le même dossier que cavea.exe. Ce fichier est disponible dans les assets de la release GitHub.
