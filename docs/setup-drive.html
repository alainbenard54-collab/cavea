🍷 Cavea — Modes de stockage

Configuration, synchronisation et icônes de l'application

Les deux modes de stockage

Mode 1

🖥️ Stockage local

La cave est stockée sur votre appareil. Aucune connexion internet requise, aucun compte externe.

Plateformes : Windows (disponible), Android local (prochainement)

Mode 2

☁️ Partage via Google Drive

La cave est partagée entre plusieurs appareils via votre Google Drive personnel.

Plateformes : Windows + Android simultanément

En Mode 2, vos données ne transitent jamais par un serveur tiers — tout passe directement entre votre appareil et votre propre Google Drive.

Choisir son mode au premier lancement

Au premier démarrage, un assistant vous propose de choisir entre les deux modes. Ce choix peut être modifié à tout moment depuis les Paramètres.

  • Mode local → vous indiquez le dossier où stocker cave.db (ou acceptez le dossier proposé par défaut)
  • Mode partagé → une page de connexion Google s'ouvre dans votre navigateur ; vous autorisez Cavea à accéder à son dossier Drive, puis choisissez si vous voulez commencer avec une cave vide ou récupérer une cave déjà présente sur Drive

Changer de mode depuis les Paramètres

Passer du Mode 1 → Mode 2

  1. Onglet Paramètres → section "Mode de synchronisation"
  2. Cliquer Activer Google Drive
  3. Authentification Google dans le navigateur
  4. Choisir : envoyer votre cave locale vers Drive, ou récupérer la cave Drive

Revenir du Mode 2 → Mode 1

  1. Onglet Paramètres → section "Mode de synchronisation"
  2. Cliquer Revenir en local
  3. L'application conserve la copie locale de cave.db et cesse toute synchronisation

Modifier le chemin de la cave (Mode 1 uniquement)

Dans les Paramètres, section "Emplacement de la cave", cliquer Modifier et sélectionner un dossier. Un redémarrage de l'application est nécessaire pour que le changement soit pris en compte.

Fonctionnement de la synchronisation (Mode 2)

Un système de verrou garantit qu'un seul appareil peut modifier la cave à la fois. Le fichier cave.db est téléchargé depuis Drive à l'ouverture et renvoyé à la fermeture.

🖥️ Windows

  • Téléchargement automatique au démarrage
  • Envoi automatique à la fermeture de l'application
  • Envoi manuel possible à tout moment via le bouton Synchroniser

📱 Android

  • Téléchargement automatique au démarrage
  • Envoi via le bouton Synchroniser en cours de session
  • ⚠️ L'envoi automatique à la fermeture n'est pas garanti sur Android (l'OS peut interrompre le processus)
Sur Android, pensez à appuyer sur Synchroniser avant de quitter l'application si vous avez apporté des modifications importantes.

En cas de lock tiers

Si un autre appareil détient le verrou quand vous démarrez l'application, Cavea vous propose de consulter votre cave en lecture seule, sans possibilité de modification. Cela protège vos données contre les modifications simultanées.

Icônes de statut dans la barre de navigation

En Mode 2, deux icônes apparaissent dans la barre de l'application pour indiquer l'état de la synchronisation.

Icône Mode Signification
🖥️ Local (Mode 1) Cave stockée localement, aucune synchronisation
☁️ Partagé (Mode 2) Cave connectée à Google Drive
🔓 Partagé — écriture Vous avez le verrou : les modifications sont possibles
🔒 Partagé — lecture seule Un autre appareil a le verrou : consultation uniquement
⏳ Synchronisation en cours Téléchargement ou envoi en cours — patienter
⚠️ Erreur de synchronisation Problème réseau ou Drive — réessayer via le bouton Synchroniser
